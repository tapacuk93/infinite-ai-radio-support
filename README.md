# Infinite AI Radio — Support

Infinite AI Radio generates an endless, ever-deepening spoken monologue on any topic you tune to, powered by Claude.

## FAQ

**How do I start listening?**
Type a topic into the tune field and tap Tune (or press Return). The broadcast begins immediately and keeps going deeper the longer you listen.

**Do I need my own API key?**
Yes. Infinite AI Radio uses your own Anthropic API key to generate the broadcast. Your key is stored only on your device and is sent only to Anthropic's API — never to any other server.

**Can I go back to an earlier part of the broadcast, or branch into a new topic?**
Yes. Use the back button to return to where you came from, or the dive button to branch deeper into a specific idea mentioned in the current segment.

**Why does the broadcast pause sometimes?**
Playback is generated and spoken segment by segment. If you pause or the app is backgrounded, generation pauses too, so you're never charged for audio you haven't listened to yet.

**Does the app work on both iPhone and Mac?**
Yes, Infinite AI Radio is available on iOS and macOS.

**I found a bug or have a feature request — where do I report it?**
Please [open an issue](https://github.com/tapacuk93/infinite-ai-radio-support/issues) on this repository, or email us directly (below).

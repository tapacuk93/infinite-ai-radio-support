# Infinite AI Radio — Support

Infinite AI Radio generates an endless, ever-deepening spoken monologue on any topic you tune to, powered by Claude. It's available on iPhone, iPad, and Mac — each device keeps its own local broadcast history (see Privacy below), so it isn't synced between them.

## FAQ

**How do I start listening?**
Type a topic into the tune field and tap Tune (or press Return). The broadcast begins immediately and keeps going deeper the longer you listen — history, an idea, a city, anything.

**Do I need my own API key?**
Yes. Infinite AI Radio uses your own [Anthropic API key](https://console.anthropic.com/settings/keys) to generate the broadcast. Usage is billed directly by Anthropic to your own account — the app itself doesn't charge for generation. Your key is stored only on your device and is sent only to `api.anthropic.com`, never to any other server. See [Anthropic's privacy policy](https://www.anthropic.com/legal/privacy) for how they handle API traffic.

**Can I go back to an earlier part of the broadcast, or branch into a new topic?**
Yes. Use the back button to return to where you came from, or the dive button to branch deeper into a specific idea mentioned in the current segment. Each dive starts its own thread, and you can always retrace your steps back up.

**Why does the broadcast pause sometimes?**
Playback is generated and spoken one segment at a time. If you pause or the app is backgrounded, generation pauses too, so you're never charged for audio you haven't listened to yet.

**Can I revisit past stations?**
Yes. A history view keeps every station you've tuned to, including how deep you dove into each one, so you can jump back in exactly where you left off.

**Are there images to go along with the broadcast?**
Yes. Infinite AI Radio looks up a relevant photo for each segment as it plays. You can toggle between text-only, image-only, and combined display modes at any time.

**Can I listen in the background, or control playback without switching back to the app?**
Yes. Audio keeps playing in the background, and you can play/pause, skip, and go back from Control Center, the Lock Screen (iPhone/iPad), the Now Playing widget in the Mac menu bar, or connected headphones — the current topic and artwork show up in Now Playing on every platform.

**Can I change the narration language?**
Yes. Switching the language takes effect immediately on the next spoken segment, with a matching high-quality voice picked automatically for that language.

**I found a bug or have a feature request — where do I report it?**
Please [open an issue](https://github.com/tapacuk93/infinite-ai-radio-support/issues) on this repository, or email us directly (below).
